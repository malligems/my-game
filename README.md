# 🧱 Brick Breaker Game

A simple and interactive **Brick Breaker Game** built using HTML, CSS, and JavaScript. The player controls a paddle to bounce a ball and break all the bricks to win the game.

---

## 🎮 Demo

🔗 Live Game:
http://amzn-game-bricks-3.s3-website.ap-south-1.amazonaws.com

---

## 📌 Features

* 🎯 Paddle control using keyboard
* ⚡ Real-time ball movement
* 🧱 Brick collision and removal
* 🏆 Score tracking system
* 🔁 Game restart functionality

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

---

## 🚀 Getting Started

### ▶️ Run Locally

1. Download or clone this repository
2. Open the project folder in VS Code
3. Open `index.html` in your browser

## 🎯 How to Play

* Use **Left Arrow (←)** and **Right Arrow (→)** keys to move the paddle
* Bounce the ball to hit and break bricks
* Don’t let the ball fall down
* Clear all bricks to win 🎉
